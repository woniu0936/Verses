Renders a list of items using ViewBinding.

Parameters

items

The data source list.

inflate

The ViewBinding inflater reference (e.g., ItemUserBinding::inflate).

key

A function to extract a stable ID for DiffUtil. Defaults to list index (not recommended for mutable lists).

span

The number of columns this item occupies in a Grid layout. Default is 1.

fullSpan

Whether this item should span the full width in Staggered layouts. Default is false.

onClick

Optional click listener for the item root view. Handled efficiently via Proxy Listener.

onAttach

Called when the view is attached to the window (enters screen).

onDetach

Called when the view is detached from the window (leaves screen).

onBind

The binding logic block.
